# Portafolio de Análisis de Ciclo de Vida (ACV) con Python

Este repositorio es una colección de proyectos y ejercicios avanzados de **Evaluación de Ciclo de Vida (LCA)**. 

## Stack Tecnológico
* **Framework:** Brightway2 (LCA en Python).
* **Fuentes de Datos:** SimaPro (Exportaciones CSV), BAFU (Base de datos Suiza), ecoinvent.
* **Análisis de Datos:** Pandas y NumPy para manipulación de matrices de inventario.
* **Visualización:** Matplotlib y Seaborn para gráficos de impacto normalizados.

---

## Proyectos Incluidos
### 1. Integración y Migración: SimaPro a Brightway
* **Descripción:** Implementación del flujo de trabajo técnico para importar inventarios complejos en formato SimaPro.csv hacia entornos de Brightway.
* **Foco:** Resolución de flujos huérfanos, mapeo de estrategias de intercambio y validación de bases de datos externas.

### 2. Comparativa de Envases: PET vs. Vidrio (Base de datos BAFU)
* **Descripción:** Evaluación del impacto ambiental de una unidad funcional de 500ml para botellas de plástico (PET) y vidrio verde.
* **Metodologías:** ReCiPe 2016 (Endpoint) e IPCC 2021.
* **Análisis:** Comparación de 18 categorías de impacto, destacando la relación entre intensidad energética de proceso y eficiencia de masa.
